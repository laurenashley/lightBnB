# lighthouseBnB

## Dependencies
- node.js
- dotenv
- nodemon
- sass
- bcrypt ^5.1.0,
- body-parser ^1.19.0,
- cookie-session ^1.3.3,
- express ^4.17.1,
- nodemon ^2.0.20,
- pg ^8.8.0 

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm run local` command.
